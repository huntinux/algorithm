# 问题
给出俩个单向链表的头指针,比如 h1,h2,判断这俩个链表是否相交。
为了简化问题,我们假设俩个链表均不带环。
问题扩展:
1.如果链表可能有环列?
2.如果需要求出俩个链表相交的第一个节点列?

# 我的分析

## 如果链表没有环
如果两个链表相交，那么从交点以后的节点一定都相同。
那么只要两个链表的最后一个节点相同，就可以确定两个链表相交

## 如果可能存在环
假设两个链表A和B，那么先判断那个链表存在环
如果：A B 都没有环,则用前面的算法即可。
如果：只有一个链表有环，那么两个链表不可能相交（）
如果：A B 都有环，那么fast指向A的第一个节点，slow指向B
的第一个节点，每次fase前进2步，slow前进1步，如果两个链表相交
那么fase一定和slow相遇（其实这与判断一个链表中是否存在环的道理是一样的。）
