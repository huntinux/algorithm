# Problem
```
编程求解:
输入两个整数 n 和 m,从数列 1,2,3.......n 中随意取几个数,
使其和等于 m ,要求将其中所有的可能组合列出来.
```

# Analysis
```
经过分析题目，发现取出的数字可以是重复的。所以思路如下
两个数的组合很好找全。两个指针，两边夹击即可。
然后可以看找到的二元组合中，哪个数字可以继续被分裂成两个数。
直到不能分裂为止
example:
n = 6 m = 8
1 2 3 4 5 6 | 8
(2, 6)
    |
	(2, 		4) 
	|			|
	(1, 1)		(1, 3) or (2, 2)
......

(3,      		5)
|        		|
(1, 2)			(2, 3) or (1, 4) 

result:
(2, 6)
(2, 2, 4)
(3, 5)
(1, 2, 5)
(1, 3, 4)
......
```

```
(3,      		5)

(1, 2)success   (2, 3)fail or (1, 4) success

(1, 2, 5)
(1, 3, 4)

```
